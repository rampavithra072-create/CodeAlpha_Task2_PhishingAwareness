# 🛡️ Phishing Awareness Training System

## 📌 Project Description

Phishing Awareness Training System is a cybersecurity awareness platform designed to educate users about phishing attacks and improve their ability to recognize and avoid cyber threats.

This project provides interactive training through phishing simulations, security awareness tips, and quizzes to help users understand common phishing techniques and follow safe online practices.

---

## 🎯 Objectives

* Create awareness about phishing attacks
* Teach users how to identify suspicious emails and links
* Improve cybersecurity knowledge among users
* Reduce the risk of falling victim to phishing scams

---

## 🚨 What is Phishing?

Phishing is a type of cyber attack where attackers trick users into revealing sensitive information such as passwords, banking details, or personal data by pretending to be a trusted source.

---

## ✨ Features

### 📊 Awareness Dashboard

* Introduction to phishing attacks
* Common phishing methods:

  * Email Phishing
  * Spear Phishing
  * Smishing (SMS Phishing)
  * Vishing (Voice Phishing)
  * Clone Phishing

### 📧 Phishing Email Simulator

* Provides sample suspicious emails
* Allows users to analyze emails
* Helps differentiate between legitimate and phishing emails

### 🔐 Security Awareness Tips

* Verify sender details
* Avoid suspicious links
* Do not share confidential information
* Check website authenticity
* Use strong passwords and security practices

### 📝 Quiz Module

* Multiple-choice cybersecurity questions
* Evaluates user awareness level
* Provides score-based feedback
